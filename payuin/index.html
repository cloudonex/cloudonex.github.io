This is a Plugin for iBilling. You must have already installed iBilling to use PayUmoney payment gateway Plugin. Please use the latest version of iBilling to use this plugin. You can download latest version by visiting codecanyon, My Account -> Downloads.

Installing

Drag & Drop Installation

  • Go to Settings -> Plugins

    Login to your iBilling. Then go to Plugins under Settings menu.

  • Upload Downloaded zip file

    Uploaded your downloaded zip file using drag and drop or click to upload. The system will automatically upload and extract the plugin.

  • Install

    You will find newly uploaded plugin in the plugin list. Simply click the Install button. The plugin will be installed and activated.

Manual Installation

  • Go to Settings -> Plugins

    Login to your iBilling. Then go to Plugins under Settings menu.

  • Extract Plugin Zip file

    Extract the downloaded file using any unzipping software. You will find a directory named "payuin".

  • Upload Plugin files in the plugin directory

    Upload the payuin folder in sysfrm/plugins/ directory using ftp or any other programme.

  • Install

    You will find newly uploaded plugin in the plugin list. Simply click the Install button. The plugin will be installed and activated.

Access PayUmoney Payment Gateway

After installing PayUmoney, you can manage this payment gateway from Settings -> Payment Gateways.

Configuration

Go to Settings -> Payment Gateways, in this page you will find list of available payment gateways including PayUmoney. Click Edit.

After clicking Edit, you will find Configuration page-


Name: Name of the Payment Gateway.
Merchant Key: Merchant key is provided by the Payment gateway.
Status: To enable / disable this payment gateway.
Merchant Salt: Merchant Salt is provided by the payment gateway. Enter this here.
Mode: Live or Sandbox. Default is Live. For testing, use sandbox.

Repositioning

To reorder Payment gateways, Go to Settings -> Payment Gateways. You will find a button named "Reorder Payment Gateway Postion." You can reorder posting by drage and drop.

Testing Payment Gateway

Before going Live, you can test by making test transaction. To enable sandbox mode, Go To Settings -> Payment Gateways. Find PayUmoney and click Edit. On next page, you will find to set mode, Live or Sandbox.

Common Troubleshooting Tips

Please make sure the plugins directory exist in the sysfrm/plugins/ folder & You are using latest version of iBilling. Also check all the files has been uploaded in the directory.

Disabling Plugin

To disable this Plugin- Go to Settings -> Plugins. You will find this plugin listed on this page. Just click Deactivate to disable this plugin

Uninstalling Plugin

After deactivating the plugin, you will be able to uninstall the plugin. To do this, simply click Uninstall button on Plugins page. Please note that, all configuration will be removed when uninstalled.
